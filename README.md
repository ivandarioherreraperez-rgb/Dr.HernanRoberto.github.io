# 🦷 Página Web Profesional - Dr. Hernán Roberto

Este es el sitio web oficial del consultorio dental del **Dr. Hernán Roberto**, diseñado para ofrecer una experiencia limpia y profesional a los pacientes.

## 🚀 Características
* **Página de Inicio:** Con diseño moderno y responsive.
* **Bloque de Información:** Detalles sobre especialidades y atención.
* **Acceso Directo:** Botón integrado para agendar citas vía WhatsApp.
* **Galería:** Imagen real del consultorio para generar confianza.

## 🛠️ Tecnologías
* HTML5 y CSS3.
* Alojado en GitHub Pages.
